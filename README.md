## 一禅
```
基于xiaomi vela开发，在xiaomi band 9 pro进行测试和维护
```

### 功能实现
- [x] 多模式计时
- [x] 专注历史查询
- [x] 通知系统
- [ ] 成就系统
- [ ] 任务系统和奖惩机制

### 特色
- 纯手绘资源
- 为应用单独设计的简洁数据库系统
- 应用内完善的动画和流畅体验


## 二次开发

### 1. 开发

```
npm install
npm run start
```

### 2. 构建

```
npm run build
npm run release
```

### 3. 调试

```
npm run watch
```
### 4. 代码规范化配置

macOS or Linux
